# Obsidian & AI Agent Collaboration Vault

このディレクトリは、ObsidianとAIエージェントが協働するために設計されています。

## ディレクトリ構造

### 1. 00_Context_Library/
**役割**: AIへの指示書、ナレッジ、フォーマットを格納する「AIの脳」です。
- `10_Knowledge/`: 会社情報、サービス仕様など、参照すべき知識。
- `20_Instructions/`: プロンプト、役割定義など、AIへの指示。
- `30_Formats/`: JSONスキーマ、スライド雛形など、出力形式の定義。

### 2. 10_Inbox/
**役割**: 一時的なメモ、整理されていないアイデア、AIによる文字起こしデータの投げ込み場所です。

### 3. 20_Production/
**役割**: コンテンツ制作のメイン作業場です。
- `10_Series/`: 連載・シリーズ企画用。
- `20_Singles/`: 単発エピソード用（年/月管理）。

### 4. 99_Archives/
**役割**: 完了したプロジェクトや過去ログを格納します。

### 5. .obsidian/
**役割**: Obsidianの設定ファイルフォルダです。
