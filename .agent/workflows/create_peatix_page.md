# Peatixイベントページ作成・公開フロー

ノンプロ研のイベントページ（Peatix）を作成する際の標準ワークフローです。
過去の成功事例（Best Practice）に基づき、集客力の高いページを効率的に作成します。

---

## 1. 準備 (Preparation)

### 1-1. 企画の確定
以下の要素が決まっているか確認してください。
- [ ] **タイトル (SEO対策):**
    - **必須:** 「無料」「オンライン」（または「Zoom」）を明記する。
    - **構成:** 前半即物、後半羅列。「イベント名」 + 「サブタイトル（検索キーワード詰め込み）」
    - Bad: `Non-Pro Ken Open Lab Vol.0` (何かわからない)
    - Good: `【無料/オンライン】ノンプロ研 オープンラボ Vol.0 ～AI/GAS/Pythonで働き方を変える公開実験室～`
- [ ] 開催日時
- [ ] ターゲット層（誰の、どんな課題を解決するか）
- [ ] タイムテーブル
- [ ] 登壇者情報

### 1-2. 画像の準備 (Assets)
- [ ] **カバー画像 (Cover Image):**
    - 推奨サイズ: 920px x 450px
    - クオリティの高い、目を引く画像を用意する。
    - 詳細は **`create_event_header`** (Event Header Image Generation Flow) ワークフローを参照してください。
    - **構図:** メインビジュアルを左右どちらかに寄せ、**「文字や顔写真を乗せる余白」**を確保すること。

---

## 2. コンテンツ作成 (Copywriting)

以下の構成テンプレート（Markdown）を使用して原稿を作成してください。

### テンプレート構造
1.  **Catch Copy:** 冒頭で「参加するメリット」や「解決する課題」を一言で（つかみ）。
2.  **Introduction:** 背景、文脈、なぜ今やるのか（共感）。
3.  **Target:** `【こんな方におすすめ】` （箇条書き）
4.  **Outline:** `【開催概要】` （日時、場所、費用）
5.  **Timetable:** `【タイムテーブル】` （詳細な流れ）
6.  **Speakers:** `【登壇者プロフィール】` （写真＋紹介文）
7.  **Organizer:** `【主催】` （ノンプロ研の説明）

---

## 3. Peatix設定 (Settings)

### 3-1. 基本情報
- **イベント名:** 検索されやすいキーワードを含める。
- **イベント詳細:** 作成したMarkdown原稿を貼り付け、太字や見出しで見やすく装飾する。
- **会場:** オンラインの場合は「オンライン開催」とし、視聴方法は「参加者へのメッセージ」や「視聴ページ」で案内する。

### 3-2. チケット設定 (Tickets)
- **種別:**
    - `無料チケット` (会員用/無料イベント用)
    - `有料チケット` (一般用)
- **枚数:** 定員を設定（少し多めに設定してもよい）。
- **販売期限:** イベント終了時刻まで、または前日まで（運用ルールによる）。

### 3-3. フォーム設定 (Form)
参加者属性を取得するためのアンケートを設定します。
- **お名前:** (必須)
- **メールアドレス:** (必須・自動取得)
- **所属・職業:** (任意)
- **認知経路:** (必須) どこでイベントを知ったか
- **期待すること/質問:** (任意)

### 3-4. 設定確認
- **カテゴリー:** 「テクノロジー／ソフトウェア」「ビジネス／キャリア」など適切に選択。
- **公開範囲:** 原則「公開」。
- **URL:** カスタムURLが設定可能な場合は設定する（例: `nonpro-openlab-vol0`）。

---

## 4. 公開と告知 (Publish & Promote)

1.  **プレビュー確認:** PCとスマホ両方で崩れがないか確認。
2.  **公開 (Publish):** 公開ボタンを押す。
3.  **拡散:**
    - Slack (`#general` / `#announcement`)
    - X (Twitter)
    - Facebook (グループ・ページ)
    - Note (イベント告知記事の埋め込み)
