---

1. [INPUT] ユーザーからイベントの詳細情報（Peatix URL、企画書、またはドラフト）を受け取る。情報が不足している場合はヒアリングする。
2. [READ] CCO (Chief Community Officer) のスキル定義ファイルを読み込む。
   - Path: `c:\Users\ntaka\nonpro-core\.agent\skills\CCO_Community_Architect\SKILL.md`
   - Purpose: 「熱量」「心理的安全性」「コミュニティへの招待」というトーン＆マナーをインストールするため。
3. [DRAFT] 以下の3つのパターンで、X（Twitter）の制限文字数（140字以内）に収まる投稿案を作成する。
   - **パターンA: Emotion（熱量重視）** - 「なぜ今やるのか」「どんな想いか」を語る。
   - **パターンB: Impact（変化重視）** - 「何が変わるのか」「これまでの常識とのギャップ」を突く。
   - **パターンC: Question（問いかけ重視）** - 読み手の思考を停止させる強い問いを投げる。
4. [ADVISE] SNS投稿の効果を最大化するためのTipsを、CCOの視点（行動経済学やコミュニティ心理学）から提案する。
   - 例: ツリー投稿の活用、画像のAltテキスト、メンション活用、投稿タイミングなど。
5. [REVIEW] 作成した案をユーザーに提示し、フィードバックをもとに修正する。
6. [SAVE] 決定した投稿案をMarkdownファイル（例: `SNS_Draft.md`）として保存する。
