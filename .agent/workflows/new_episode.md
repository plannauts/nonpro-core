description: Create a new podcast episode directory and initial files
---
1. Ask the user for the "Series Name" (e.g., Neighbor_AI_Strategy) and "Episode Name" (e.g., Ep03_Title_Here).
2. Create the directory: `20_Production/10_Series/[Series Name]/[Episode Name]`
3. Create an empty `comments.md` file in that directory.
   - Initial content: `# リスナーコメント（コミュニティ・タイム用参照資料）\n\n- (ここにコメントを貼り付けてください)`
4. Create an empty `script.md` file in that directory.
   - Initial content: `# Podcast Script: [Episode Name]\n\n(Run podcast_script_maker instruction to generate content)`
5. Notify the user that the folder and files are ready.
