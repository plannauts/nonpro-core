# コミュニティ3.0

## 定義
「ギブ（Give）」を基点とした、自律的かつ共創的な学習コミュニティの進化形態。

## 背景・文脈
コミュニティのあり方を段階的に定義したもの。

*   **1.0 (ファンクラブ型):** 特定のリーダーやカリスマを中心とし、参加者はコンテンツを受け取る「消費者」である状態。
*   **2.0 (サロン型):** 参加者同士の交流が行われるが、主な価値は「居場所」や「人脈」にあり、学習や共創は副次的。
*   **3.0 (自律共創型):** 「先生」と「生徒」の境界がなく、全員が教え合い、学び合う状態。参加者自身がコンテンツを生み出し、コミュニティを運営する当事者（当事者意識）を持つ。「教えることは二度学ぶこと」の実践の場。

ノンプロ研が目指し、体現しているモデル。
