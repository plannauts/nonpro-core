# 教えることは二度学ぶこと

## 定義
アウトプット（他者に教えるプロセス）こそが、最も効果的なインプット（自身の学習）であるという学習原則。

## 背景・文脈
フランスのモラリスト、ジョゼフ・ジュベール（Joseph Joubert）の言葉 "To teach is to learn twice." に由来する。

ノンプロ研における学習サイクルの中核をなす概念。
1.  **Input:** 学ぶ。
2.  **Practice:** 手を動かす。
3.  **Output (Teach):** 学んだことをブログに書く、LTで話す、講座で教える。

人に教えるためには、曖昧な理解を明確にし、論理だてて言語化(再構築)する必要があるため、単に聞くだけの学習よりも定着率が圧倒的に高くなる（ラーニングピラミッド参照）。
