# 資産ゼロ (Asset Zero)

## 定義
AIの出力を理解せずにコピペし続けることで、自分自身の脳内やスキルセットに知識・経験が一切蓄積されない状態。

## 背景・文脈
便利さ（タイパ）を追求してAIに丸投げした結果、費やした時間に対して将来の競争力が形成されない「徒労」を表す。
「The Messenger（AIの伝書鳩）」が陥る典型的な罠。
AIのモデルがアップデートされたり、サービスが終了したりした瞬間に、その人の市場価値も消滅するリスクがある。
