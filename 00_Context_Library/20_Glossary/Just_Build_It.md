# Just Build It

## 定義
「自分の手で作る」という行為そのものに価値を見出し、ブラックボックスへの依存を拒絶する態度のこと。ノンプロ研の美学（Aesthetics）。

## 背景・文脈
Nikeの "Just Do It" へのオマージュ的なニュアンスを含む。
「AIにやらせれば楽ができる」という誘惑に対し、「あえて自分で作るプロセスを楽しむ」「中身を理解して支配する」ことが **Cool** であり、依存することは **Lame**（ダサい）であるという価値観を表明する言葉。
これが「デジタル主権」を守るための精神的防壁となる。
