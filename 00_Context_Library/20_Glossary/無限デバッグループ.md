# 無限デバッグループ

## 定義
AIが生成したコードのロジックを理解していないため、エラーが発生した際に適切な修正ができず、AIへの再質問とコピペを延々と繰り返す状態。

## 背景・文脈
「99%まではAIが行けるが、最後の1%（ラストワンマイル）は人間しか埋められない」という現実により発生する。
基礎知識（構文・構造）がない場合、このループから抜け出すことができず、最終的に「直せないコード」という技術的負債を抱え込むことになる。
タイパを求めた結果、逆にタイパが破綻する典型例。
