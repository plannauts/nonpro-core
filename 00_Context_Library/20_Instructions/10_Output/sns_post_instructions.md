# SNS投稿作成インストラクション

## 入力として与えられるもの
- note記事（Markdown形式）

## 目的
X、Instagram、Facebookに適した投稿と、Podcastの概要文、noteのハッシュタグを作成します。
**全体的に、単なる告知ではなく、読者の「感情（焦り、モヤモヤ）」に寄り添い、解決策を提示する「共感型・ストーリー型」の投稿を目指します。**

## 作成物と要件

### 1. 朝7:10のX＆Facebookへの投稿（共通）
- **役割**: モーニング・コーヒータイムに読むエッセイのような立ち位置。
- **構成**:
    1. **共感フック**: 読者が抱えている潜在的な悩みや感情を言語化。
    2. **安心感の提示**: 「あなただけじゃない」とデータ等で安心させる。
    3. **解決の提示**: 記事（処方箋）への誘導。
- **重要**: **X（旧Twitter）の制限（140文字）に収まるように作成してください。**
- **ハッシュタグ**: X用に合わせて **2個まで**。`#隣ラジオ` 必須。

### 2. 昼12:10のX、Facebook、Instagramへの投稿
インフォグラフィック画像（ユーザー作成）と併せて投稿することを前提とした、各プラットフォームに最適化されたテキスト。

#### A. X（旧Twitter）用
- **特徴**: 短く、パンチを効かせる。リスト形式を活用。
- **構成**:
    - ターゲットへの呼びかけ。
    - 結論・正体の提示。
    - 箇条書き（1️⃣ 2️⃣ 3️⃣）で要点を可視化。
    - オチ・解決策のソース提示。
    - 図解への誘導。
- **重要**: **必ず140文字以内に収めてください**（URL分やタグを含めた余裕を持つこと）。
- **ハッシュタグ**: **2個まで**。`#隣ラジオ` 必須。

#### B. Facebook用
- **特徴**: しっかり読ませる「手紙」のようなスタイル。
- **構成**:
    - タイトル風のキャッチコピー。
    - 挨拶。
    - 丁寧な日常会話風の導入。
    - 内面の吐露、解説、結論と誘導。
- **長さ**: 400〜600文字程度。
- **ハッシュタグ**: **3個程度**。`#隣ラジオ` 必須。

#### C. Instagram用
- **特徴**: 改行を多用したポエム調・ビジュアル重視の構成。
- **フォーマット**:
    - 行間を空ける（`.` で改行するなど）。
    - 問いかけから入る。「実は…」という展開。
    - アクション誘導（「スワイプしてチェック👉」）。
    - プロフィールリンクへの誘導。
- **ハッシュタグ**: **3～5個**。投稿の最後に記載。`#隣ラジオ` 必須。

### 3. Podcastタイトル
- **フォーマット**: `#[連番] [記事タイトル] 【[シリーズ名][シリーズ番号]】`
- **例**: `#0035 AI時代に「自律」という武器を手に入れる〜ニュースに振り回されないための生存戦略〜【隣のAI戦略02】`
- ※連番は別途管理のエピソードログ等を参照またはユーザーに確認すること。

### 4. Podcast概要文
- エピソードの内容を要約した紹介文（150文字程度）の後に、**以下の固定フッターを必ず追加してください。**

```text
▼【今日の放送を「テキスト」で読む（note）】<br>
[note URL]<br>
▼【もっとつながろう！無料のコミュニティに参加する（Discordコミュニティ）】<br>
https://discord.gg/8f926kut2B<br>
▼【番組への感想・ご意見はこちら】<br>
X（旧Twitter）でハッシュタグ #隣ラジオ をつけてポストしてください<br>
---<br>
▼【学習コミュニティ「ノンプログラマーのためのスキルアップ研究会」（ノンプロ研）の紹介はこちら】<br>
https://plannauts.co.jp/nonproken/<br>
▼【ノンプロ研講座はこちら】<br>
https://plannauts.co.jp/nonproken-course/
```

### 5. noteのハッシュタグ
- note投稿時に設定するハッシュタグのリスト。
- **出力形式**: 説明書きは不要。ハッシュタグのみをスペース区切りで出力。
- **数量ルール**: 全体で5~8個（大カテゴリ1~2個、中カテゴリ2~3個、記事のキーワード1~2個）。
- 必須タグ: `#隣ラジオ`
